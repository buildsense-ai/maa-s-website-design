# MaaS Website Design

## 功能说明
- Hero 采用 60/40 左文右视觉布局与 Prop Mesh 动效。
- 首屏背景由深色渐变过渡到浅色基础设施区。
- 标题使用白到浅灰的纵向渐变，副标题低对比度呈现。
- Header 主 CTA 随进入浅色区域切换为深靛青配色。
- 深浅过渡区域加入合作方占位 Logo 与滚动展示。
- Header 导航改为多页面路由，支持 Product/Resources 下拉。
- 新增 Writer Agent、Documentation、Pricing、Company 页面。

## 更新说明
- Hero 文案与布局重构为 60/40 左对齐结构。
- 新增 Three.js Prop Mesh 动效与交互激活。
- 首屏深浅过渡与浅色区层叠样式完善。
- Header CTA 增强发光并在浅色区切换配色。
- 移除临时的 three 类型声明文件，回归官方类型包。
- 新增多页面导航与产品/资源下拉菜单。
- 增加 Writer Agent 详情页并集成架构图。
- 增加 Documentation、Pricing、Company 等页面占位内容。
